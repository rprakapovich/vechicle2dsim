# vechicle2dsim
After start vehicle2dsim.m script you must choose one of the existing maps by open File menu and press on Open button.
You should see mobile robot in 2D environment with geometric primitive obstacles. Mobile robot has two types of sensor system: 8 binary corner zones of safety and 2D lidar. Lidar can be configured.
Next in Manual mode you can control to moving of mobile robot by virtual or real key buttons.
You can choose Autonomous mode in which mobile robot must moving according choosed control script.
